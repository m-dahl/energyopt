EnergyOpt unix distribution.
============================

Dependencies
============
* ActiveMQCpp - http://activemq.apache.org/cms/
* RapidJson - https://github.com/miloyip/rapidjson [HEADERS INCLUDED]
* IpOpt (coinor-libipopt-dev)
* ADOLC (libadolc-dev)

Installation & Usage
====================
* make
* ./energyopt busaddress [port]
